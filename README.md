#  Приключенческая игра: Путь к сокровищам

Добро пожаловать в текстовое приключение, где тебе предстоит исследовать загадочный лабиринт, собирать предметы и искать путь к сундуку с сокровищами.  
Каждая комната хранит свои тайны — будь внимателен, думай логически и используй найденные предметы с умом.  

---

##  Описание проекта

Игра представляет собой консольное приключение, в котором игрок:
- перемещается между комнатами, используя команды (например, `go north` или `take torch`);
- собирает и использует предметы;
- решает небольшие головоломки;
- стремится найти сундук с сокровищами — **главную цель игры**.

Проект написан на Python и имеет модульную структуру:
- `constants.py` — описание комнат, предметов и логики переходов;  
- `player_acions.py` — главный игровой цикл;  
- `utils.py` — вспомогательные функции;  
- `main.py` — точка входа для запуска игры.

##  Установка

### Вариант 1 — с использованием Poetry

1. Убедись, что у тебя установлен [Poetry](https://python-poetry.org/).  
2. Клонируй репозиторий и перейди в папку проекта:
   ```bash
   git clone https://github.com/username/adventure-game.git
   cd adventure-game
3. Установи зависимости
    poetry install

### Вариант 2 - с использованием Makefile

Если у тебя установлен make, просто выполни:
    make install

## Запуск игры
1. Через Poetry
    poetry run project
2. Через Makefile
    make project
